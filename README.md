## Run Locally

Clone the project

```bash

git clone https://github.com//github-assignment.git

```

Go to the project directory

```bash

cd github-assignment

```

Install dependencies

```bash

npm install

```

Install both client and server side dependencies with npm

```bash

npm run getpkg

```

Start the server

```bash

npm start

```
